# Bank Management System

A simple Command Line Bank Management System built using Python. This project is based on Object-Oriented Programming (OOP) and uses JSON file handling to store account data.

## Features

- Create Account
- Show Accounts
- Deposit Money
- Withdraw Money
- Transfer Money
- Transaction History
- Save & Load Data using JSON
- PIN Verification

## Technologies Used

- Python
- OOP
- JSON

## How to Run

```bash
python main.py
```
